## 大按钮
大按钮高度固定为 88px（44pt），圆角大小固定 8px（4pt）。
自定义配置：按钮文案（不建议超过 8 个中文字符）。

## 小按钮
小按钮高度固定为 56px（28pt），宽度最小 114px（57pt），圆角大小固定 6px（3pt）。
按钮内距离文字最小 30px（15pt），文字较长时，按钮可按所需宽度拉伸适配。
