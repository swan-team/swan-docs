### swan.getMenuButtonBoundingClientRect()

> 基础库 swan-game.js 1.12.0 版本开始支持。

获取菜单按钮（右上角胶囊按钮）的布局位置信息。坐标信息以屏幕左上角为原点。  
**返回值：** Object  

|属性|类型|描述|
|-|-|-|-|
|width|number|宽度|
|height|number|宽度|
|top|number|上边界坐标|
|right|number|右边界坐标|
|bottom|number|下边界坐标|
|left|number|左边界坐标|
