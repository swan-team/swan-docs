**解释**：用于展示百度 App 开放的数据。

##  属性说明 


| 属性名 | 类型     | 默认值  | 必填 |说明              |
| :--- |: ------ | :---- | :---- |:---- |
| type | String | | 是 |开放数据类型 |

###  type 有效值 

| 值 | 说明 |
|:--- |:----- |
| userNickName | 用户昵称 |
| userAvatarUrl | 用户头像 |
| userGender | 用户性别 |
