> 使用顺序：setShadow(offsetX, offsetY, blur, color)

**解释**：设置阴影样式。

 
## 方法参数

 

|参数名|属性|必填|默认值|说明|
|----|----|----|---|---|
| `offsetX`|Number|是|| 阴影相对于形状在水平方向的偏移。|
| `offsetY`|Number|是||阴影相对于形状在竖直方向的偏移。|
| `blur `|Number|是||阴影的模糊级别，数值越大越模糊，范围：0 ~ 100。 |
| `color`| [Color](/develop/api/canvas_color/) |是||阴影的颜色。 |
