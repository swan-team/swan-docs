**解释**：对当前坐标系的原点 (0, 0) 进行变换，默认的坐标系原点为页面左上角。

 
## 方法参数 

 
|属性名 |类型  |必填 | 默认值 |说明|
|:---- |:---- |:---- |:----|:----|
| `x`|Number|是||水平坐标平移量 |
| `y`|Number|是||竖直坐标平移量|

### 代码示例 



```js
const canvasContext = swan.createCanvasContext('myCanvas');

canvasContext.strokeRect(10, 10, 150, 100);
canvasContext.translate(20, 20);
canvasContext.strokeRect(10, 10, 150, 100);
canvasContext.translate(20, 20);
canvasContext.strokeRect(10, 10, 150, 100);

canvasContext.draw();
```
