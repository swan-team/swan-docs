**解释**：画一个矩形(非填充)。

 
## 方法参数 

 
|属性名 |类型  |必填 | 默认值 |说明|
|:---- |:---- |:---- |:----|:----|
| `x`|Number |是||矩形路径左上角的 x 坐标。|
| `y`|Number |是||矩形路径左上角的 y 坐标。|
| `width`|Number |是||矩形路径的宽度。|
| `height`|Number |是||矩形路径的高度。|

### 代码示例

```js
canvasContext.setStrokeStyle('blue');
canvasContext.strokeRect(30, 30, 150, 75);
canvasContext.draw();
```
