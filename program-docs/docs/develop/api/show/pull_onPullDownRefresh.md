### 代码示例 



* 在 swan 文件中

```html
<view class="container">
    <view class="card-area">
        <view class="page-info">下拉页面即可刷新</view>
    </view>
</view>
```

* 在 js 文件中

```js
Page({
    data: { },
    onPullDownRefresh(){
        swan.showModal({
            title: 'success',
            content: '已触发页面刷新'
        });
    }
});
```
