**解释：** 顶部导航bar可以自定义

##  属性说明 

|属性名 | 类型 | 必填 | 默认值 |说明 |
|---|---|---|---|---|
|backgroundColor |string |否|#fff|导航背景色|

###  代码示例

* 在 swan 文件中

```html
<smt-navigation>
    <view>
        <text>{{title}}</text>
        <text>{{subTitle}}</text>
    </view>
</smt-navigation>
```


* 在 js 文件中

```javascript
Page({
    data: {
        title: '主标题',
        subTitle: '副标题'
    }
});
```
