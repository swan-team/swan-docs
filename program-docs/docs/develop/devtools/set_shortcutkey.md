### 项目

|Mac OS 快捷键 | Windows 快捷键 | 说明 |
|---|---|---|
| ⌘ + Q | ctrl + Q | 退出开发者工具 |
| ⇧ + ⌘ + N | shift + ctrl + N | 新建项目 |
| ⇧ + ⌘ + W | shift + ctrl + W | 关闭当前项目 |

### 文件

|Mac OS 快捷键 | Windows 快捷键 | 说明 |
|---|---|---|
| ⌘ + N | ctrl + N | 新建文件 |
| ⌘ + S | ctrl + S | 保存当前文件 |
| ⇧ + ⌘ + S | shift + ctrl + S | 保存所有文件 |
| ⌘ + W | ctrl + W | 关闭当前文件 |
