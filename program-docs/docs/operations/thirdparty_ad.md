### 一、协议的范围
1.1 本协议是您与百度之间关于在百度小程序平台进行网盟广告合作所订立的协议。“百度”是指百度公司及其相关服务可能存在的关联运营主体。“用户”是指注册、登录、使用百度小程序的组织，在本协议中更多地称为“您”。“其他用户”是指包括其他除用户本人外与百度小程序平台服务相关的用户。

1.2 <u>**本协议内容同时包括百度可能不时发布的与百度小程序平台规则、服务声明、业务规范、通知及公告指引等内容（以下统称为“专项规则”）。上述内容一经正式发布，即为本协议不可分割的重要组成部分，您同样应当遵守。**</u>

### 二、开通条件要求
2.1 您应当是具备独立合法有效的法人资格，在您经营所在当地工商局正式注册并取得营业执照的企业或其他经国家机关合法认证的具备独立主体资格的主体，且具有合法的对公账户。

### 三、合作内容
3.1 您可以通过百度小程序平台提供的功能设置，在您的百度小程序上开发广告资源位，接入百度网盟广告对外展示并获得相应的收益。

### 四、账号权益管理与分配
4.1 <u>**如果您的百度小程序账号由其他主体代为运营管理（以下称为“第三方运营主体”），百度有权视您与第三方运营主体构成关联关系。第三方运营主体在您的账号下进行的所有操作将视为已获得您的充分、有效、明确的授权，所产生的任何后果将由您与第三方运营主体连带承担。您同意本协议即代表您同意百度有权按照平台规则将您在百度小程序平台上的网盟广告收益全部直接支付给第三方运营主体，且第三方运营主体有权根据收款金额直接向百度开具发票。您有权与第三方运营主体对网盟广告收益进行分配，但您双方就收益分配产生的任何纠纷，由您双方自行解决，百度不承担法律责任。同时，基于您自愿选择将收取收益的权利、开票的义务等相关合同权利义务授予第三方运营主体并与其达成一致，故您和第三方运营主体同意就小程序的设立、运营、收益分配、义务履行等合同权利义务连带向百度承担法律责任，且您和第三方运营主体之间就收益分配以及任何其他方面产生的任何纠纷，不得作为不履行连带法律责任的抗辩理由。**</u>

4.2 <u>**如果您的百度小程序账号由您自行运营管理，您的账号下进行的所有操作产生的任何后果，将由您自行承担。百度将按照平台规则将网盟广告收益支付至您的对公账户。**</u>

### 五、关联关系解除
5.1 您与第三方运营主体中的任何一方可向对方发出通知解除关联关系，关联关系自双方确认或通知发出48小时后（以二者中时间较早者为准）自动解除。

5.2 平台可根据平台的协议/规范/不定期发布的规则/通知等，对不符合要求的账号解除关联关系。

5.3 <u>**您双方解除关联关系后，您需要向百度发出解除关联关系的通知，百度将在收到通知后按照平台规则将网盟广告收益直接支付至您的对公账户。**</u>

5.4 <u>**解除关联关系后，您应及时变更账号密码并完善账号的各项信息，避免造成不必要的损失。如产生任何损失，您需要自己承担，百度不承担任何责任。**</u>

### 六、责任的承担
6.1 <u>**您理解并同意：您与第三方运营主体之间的任何纠纷，均由您双方自行解决并承担相应责任。如百度因此遭受任何损失或权利主张，您应向百度赔偿。**</u>

6.2 <u>**关联期限内，您与第三方运营主体的行为承担连带责任。任何一方违反本协议、《百度小程序网盟广告合作协议》或百度的其他规则，尤其是存在恶意注册、违规点击及其他对百度推广客户权益造成伤害、对用户体验造成损害、对广告联盟市场合作秩序造成扰乱、对百度各产品及百度品牌商誉造成不利影响、对百度构成不正当竞争或侵犯百度合法权益的行为时，平台可选择以下一项或多项处理措施：责令账号整改/降低账号功能权益/扣除违规收益/其他处理措施。如您的账号给百度或第三方带来不利影响的，您还应向百度或其他第三方承担连带赔偿责任。**</u>

6.3 <u>**解除关联关系的，并不免除在解除关联关系之前已经产生的行为的关联责任。**</u>

### 七、合作终止
7.1 <u>**百度小程序或百度网盟广告在发生不可抗力/互联网特殊行业发展/产品、业务发展需求变更的情况下，可变更、终止该产品模式，该等变更/终止的通知将以站内信或其他方式送达给您，自通知发出后，本协议自动解除。百度小程序将给予您一定的缓冲期，以供您处理合作终止后的事宜。**</u>
