**networkType 的合法值**

`Object` 类型的对象：

|-|-|
|wifi|wifi 网络|
|2g|2g 网络|
|3g|3g 网络|
|4g|4g 网络|
|unknown|Android 下不常见的网络类型|
|none|无网络|
