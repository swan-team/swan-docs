**解释**：同步获取当前 storage 的相关信息。

**方法参数**： 无

**示例**：

* 在 js 文件中

```js
try {
    const result = swan.getStorageInfoSync();
    console.log('getStorageInfoSync success', result);
} catch (err) {
    console.log('getStorageInfoSync fail', err);
}
```

* Andriod

|错误码|说明|
|--|--|
